


## 概述

ONT ID 为每个用户、公司、物品等任何实体建立基于密码学的数字身份，数字身份基于本体区块链技术，不受制于任何中心化机构，完全由用户自己掌控，并具有安全、可信的特点。

你第一次拥有了基于区块链的数字身份 ONT ID。为了让你的ONT ID 更可信，更多源，你可以通过不同领域的信任源对 ONT ID 进行认证和背书，如 eID、CA、政府、机构、学校、公司、社群、个人等。认证授权后，你将拥有不同身份信任源的区块链证书（即区块链可信声明）。
